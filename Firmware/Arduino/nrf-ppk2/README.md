# ESP32 Power monitoring with Nordic PPK2

Follow the [tutorial](https://github.com/CampusIoT/tutorial/tree/master/nrf-ppk2#readme) : select 3.3V from the supply voltage.

## Pinout

| PPK2 | ESP32 |
| ---- | ----- |
| `VOUT`| `3V3`|
| `GND` | `GND`|
| `D0`  | `D0` |
